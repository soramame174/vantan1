# MelodyNotes (メロディノ)

「誰でも自由に音楽を楽しみ、創作活動を支援する」ことを目的としたフリー音楽WEBアプリケーションです。

## 概要
MelodyNotesは、音楽クリエイターが自作楽曲をフリー素材として登録・公開し、他のユーザーやクリエイターがそれらを自由に検索・利用できるプラットフォームです。動画制作、ゲーム開発、広告制作など、様々なシーンで利用可能な音楽とクリエイターを繋ぐ架け橋となります。

## 主な機能

### 1. 楽曲登録・管理機能
* **フリー公開・非公開設定**: プロフィールのみに表示する非公開設定と、全体公開を選択可能。
* **充実したメタデータ**: タイトル、URL、曲ファイル、画像、歌詞、ジャンル、キーワードを設定可能。

### 2. 高度な検索機能
* **キーワード検索**: タイトル、ジャンル、アーティスト名などから目的の楽曲を即座に発見。
* **トレンド把握**: よく検索される楽曲から、現在のトレンドを把握可能。

### 3. プレイリスト（フォルダ）機能
* **自分だけの管理**: お気に入りの楽曲をフォルダ分けして整理。
* **並べ替え機能**: 好きな順番に並べ替えて管理可能。
* **連続再生**: フォルダ内の楽曲を上から順に自動再生。
* **共有機能**: 自分が作成したプレイリストを他ユーザーと共有し、オススメを紹介可能。

### 4. リクエスト機能
* クリエイター同士のアイデア共有や、楽曲ニーズのやり取りを促進し、新たな創作活動を支援。

## 他社サービスとの差別化ポイント
* **フォルダ管理のしやすさ**: 直感的な操作でプレイリストを構築可能。
* **共有・コミュニケーション**: プレイリスト共有を介して、個人の「推し」やオススメを広めやすい設計。
